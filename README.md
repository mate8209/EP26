# Economía Política - Landing page GitHub Pages

Sitio estático generado a partir de la presentación `Clase Inicial (2_6_26).pptx`.

## Estructura

```
index.html
styles.css
script.js
assets/slides/
```

## Publicación en GitHub Pages

1. Crear un repositorio público en GitHub.
2. Subir todos los archivos y carpetas de este ZIP.
3. Ir a Settings > Pages.
4. Elegir Deploy from a branch.
5. Seleccionar branch `main` y carpeta `/root`.
6. Guardar.

La web quedará disponible en `https://TUUSUARIO.github.io/NOMBRE-DEL-REPOSITORIO/`.

## Ajuste recomendado

Revisar fechas y días de semana antes de publicar, especialmente el primer parcial indicado como 3/7 (Martes).
