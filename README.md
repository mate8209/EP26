## Publicación en GitHub Pages

1. Crear un repositorio nuevo en GitHub.
2. Subir todos los archivos respetando la estructura anterior.
3. Ir a **Settings → Pages**.
4. En **Build and deployment**, seleccionar **Deploy from a branch**.
5. Elegir la rama `main` y la carpeta `/root`.
6. Guardar. GitHub generará la URL pública del sitio.

## Ajustes sugeridos antes de publicar

- Revisar la consistencia entre fechas y días de la semana.
- Reemplazar capturas de diapositivas por imágenes originales si se desea una versión más liviana.
- Agregar logo institucional oficial si corresponde.
